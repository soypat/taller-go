Qué es Go?

Lenguaje de programación moderno caracterizado por su simplicidad, manejo de memoria (garbage collection) y buen manejo en equipos grandes.

Mientras esperaban 40 minutos para que termine de compilar un programa C++, un equipo de 3 personas de Google se pusieron a pensar que forma tendría un lenguaje moderno...

    memes["carpincho"] = 2020

	// DEL_S OMIT

Se puede eliminar un conjunto clave-valor con delete

Switch

        
El switch es la estructura de control más flexible de Go y se usa para diversos casos.





//+build ignore OMIT

package main

import "fmt"

func main() {

    i := 2

    switch i {
    case 2:
        fmt.Println("Dos")
    case 3:
        fmt.Println("Tres")
    case 4, 5:
        fmt.Println("Cuatro o cinco")
    default:
        fmt.Println("No conozco ese número")
    }



}



    

  • Sirve de else if simplificado
    • 

  • Verificación de tipo (Type checking)
    • 

  • Y muchos más
    •

Paquete os, manejo de errores y lectura de archivos

        



package main

import (
    "fmt"
    "io/ioutil"
    "os"
)

func main() {
    f, err := os.Open("archivo.txt")
    if err != nil {
        fmt.Println("No pude abrir el archivo:", err)
        // termina el programa indicandole al sistema operativo que hubo un error
        os.Exit(1)
    }
    buff, _ := ioutil.ReadAll(f)
    fmt.Println(string(buff), "\nArchivo leido!")
}
